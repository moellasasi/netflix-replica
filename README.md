# netflix-replica
netflix clone- a complete replia of netflix as sasiflix
A full-stack Netflix clone that replicates the core design and functionality of the Netflix platform. This project includes a responsive frontend, secure backend, and dynamic content rendering using modern web technologies. It demonstrates full-stack development skills including authentication, API integration, and scalable architecture.
Features
🎬 Netflix-style responsive user interface
🔐 User authentication (Login / Signup / Logout)
📺 Dynamic movie and TV show data from backend/API
🧾 User session management and protected routes
❤️ Add to My List / Favorites functionality
🔎 Browse movies by categories and genres
⚡ Fast and interactive frontend with optimized performance
🌐 REST API integration between frontend and backend
